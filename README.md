## Features
* Uses GLUT, GLEW.
* Compiled with GCC using `-lglut -lGL -lGLU -lGLEW`
* Collision detection
* Variable movement speed by handling keypress events

![nknkn](./ss.png)

## Usage

#### Clone and build:
```bash
git clone "https://github.com/tusharpandey13/cgproject.git"
cd cgproject
make && chmod +x ./main
```
#### Running:
```bash
./main
```
